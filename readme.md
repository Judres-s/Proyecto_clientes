Pasos a realizar.

1- Descargar la carpeta venv para realizar la actividad en un entorno virtual
2- Realizar el siguiente comando en la terminal "Python -m venv venv" Luego de esto esperar la descarga de los archivos
3- Revisar en la carpeta "Scripts" el archivo "activate" si no se encuentra no se podrá continuar.
4- Activar el entorno virtual con el siguiente comando ".\venv\Scripts\activate"
5-Luego, instalaremos el framework con el siguiente comando en la terminal "pip install "fastapi[standard]"
6-Al instalar el framework lo siguiente que tendremos que realizar es crear un archivo llamado "main.py" y realizar el código para el entorno virtual y poder ejecutarlo.
7- Luego de haber creado el código con su respectiva librería de FastAPI y de haber creado los endpoints, finalmente se ejecuta el entorno virtual.
8- Ejecutamos nuestra aplicación de fastapi de la siguiente manera, en la terminal escribimos: "fastapi dev main.py" decimos "fastapi" para ejecutar la herramienta de fastapi, "dev" como modo de desarrollo y "main.py" como nuestro archivo.
